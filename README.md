🌐 Website Traffic Analysis Dashboard (Power BI + SQL)

A comprehensive Website Traffic Analysis Dashboard built using Power BI with data sourced and transformed via SQL. This project delivers actionable insights into user behavior, engagement patterns, and traffic performance across multiple dimensions including devices, regions, and acquisition channels.

📊 Executive Overview

This dashboard enables data-driven decision-making by providing a 360° view of website performance. It is designed for digital marketers, product teams, and business stakeholders to:

Monitor user engagement trends
Identify drop-off points and optimization opportunities
Evaluate marketing channel effectiveness
Understand audience distribution across geography and devices
🎯 Business Objectives
Improve user engagement and retention
Optimize traffic acquisition strategies
Identify high-performing regions and segments
Reduce bounce rates and session drop-offs
Enhance device and browser experience
📈 Key Metrics
Session Duration: Total and average time users spend on the website
Bounce Rate: Percentage of users leaving after a single interaction
Page Views: Total number of pages viewed across sessions
Traffic Sources: Performance across organic, paid, social, email, and referral channels
Device Distribution: Engagement split across desktop, mobile, and tablet
Geographical Metrics: Regional and city-level performance insights
